# Shadow Texture

**IMPORTANT:** Place your `shadow.png` file in this directory (`public/textures/shadow.png`).

The texture should be:
- Large (>= 2048px)
- Soft, transparent vignetting
- Light grain
- Subtle, not harsh

The overlay will automatically use this texture once the file is placed here.

**Current opacity:** 0.4 (40%) - adjust in `src/App.tsx` line 147 if needed.
